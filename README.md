Bringing this specific Serverless project into public status with the idea to share and grow in the future, this project is a pilot of many that would serve as feedback and encourage more projects to come to public side as open-source projects.


This project is intended to build a full cloud based serverless arrchitecture project, allowing developers and interested domaine folks to use the Infrastructure as Code (IaC) to build a "Portable" sustanable project from scratch that also supports the pay-as-you-go concept.

Technology wise , this project is based on:
-AWS cloud
-Simplified production grade RAG project
-Github actions and CI/CD
-Event based architecture and asynchronous communication between components.
